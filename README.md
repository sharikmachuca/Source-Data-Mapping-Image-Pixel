# Source-Data-Mapping-Image-Pixel

**Descripción:** el repositorio presenta una actividad académica orientada a comprender cómo los sistemas de información pueden transformar datos provenientes de fuentes no estructuradas en datos organizados. A partir de una imagen, se realiza un ejercicio de conversión y almacenamiento de información visual en un formato estructurado que permite su posterior lectura y procesamiento.

Este tipo de ejercicios permite entender cómo los datos pueden capturarse, transformarse y almacenarse, replicando de forma simplificada procesos que ocurren en sistemas de información empresariales. En un contexto empresarial, actividades como esta son relevantes porque muchas organizaciones trabajan con grandes volúmenes de información no estructurada, como imágenes, documentos o archivos multimedia. Convertir estos datos en información estructurada facilita su almacenamiento, análisis y uso en procesos de negocio, como control de calidad, análisis de productos, automatización de procesos o desarrollo de sistemas de analítica e inteligencia artificial.

**Objetivo:** ilustrar cómo una fuente de datos visual (imagen) puede convertirse en datos estructurados y almacenarse en un repositorio que permita su posterior procesamiento y reconstrucción.

**Proceso:** el flujo del proyecto se desarrolla en tres iteraciones explicadas a continuación.

_Iteración 1_
- Lectura de la imagen.
- Extracción de píxeles.
- Conversión de colores a formato hexadecimal.
- Creación de un archivo Excel con celdas coloreadas.
  
_Iteración 2_
- Lectura de un archivo Excel con información de colores.
- Extracción de los códigos de color.
- Reconstrucción de la imagen en un nuevo archivo Excel.

_Iteración 3_
- Conversión de la imagen en una matriz de valores RGB.
- Almacenamiento de los valores RGB en Excel.
- Lectura de la matriz RGB y reconstrucción visual de la imagen.

**Resultado y utilidad:** el resultado del proyecto es la creación de un repositorio de datos estructurados a partir de una fuente visual. La imagen se transforma en una matriz de valores RGB que puede almacenarse, procesarse y reutilizarse posteriormente para reconstruir la información visual original.

Este tipo de procesos refleja cómo los sistemas de información empresariales convierten datos no estructurados en información organizada que puede ser utilizada para análisis y toma de decisiones. En entornos reales, esta lógica es utilizada para el procesamiento de imágenes, reconocimiento de patrones, análisis visual de productos o automatización de tareas dentro de plataformas digitales y sistemas de analítica empresarial.
